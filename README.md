# 🚀 Freelance Marketplace для RuStore

**Статус:** v0.2 - Backend + Flutter готовы 🚀  
**Дата релиза:** 20 июня 2026

Мобильная платформа для фрилансеров и заказчиков. Платежи идут напрямую между пользователями. Монетизация через премиум профили и продвижение объявлений.

---

## 📦 Структура проекта

```
freelance-app/
├── backend/                      # Node.js API
│   ├── server.js                 # Entry point
│   └── package.json
├── flutter_app/                  # Android приложение (Flutter)
│   ├── lib/
│   │   ├── main.dart
│   │   ├── providers/            # State management
│   │   └── screens/              # UI экраны
│   └── pubspec.yaml
├── database/
│   └── DATABASE_SCHEMA.sql       # PostgreSQL schema
└── docs/
    ├── ARCHITECTURE.md           # Архитектура и техплан
    ├── SETUP.md                  # Инструкция по запуску
    └── API.md                    # Документация API (TODO)
```

## ⚡ Быстрый старт

### Backend
```bash
cd backend
npm install
# Создать .env (см. docs/SETUP.md)
npm run dev
```

### Flutter
```bash
cd flutter_app
flutter pub get
flutter run
```

## 🎯 MVP Функции

### ✅ Реализовано
- Авторизация (регистрация/вход)
- Каталог заказов с фильтром
- Профиль пользователя
- Структура Backend API
- PostgreSQL schema

### 📍 В разработке
- In-app чат
- Интеграция Яндекс.Касса
- Система рейтингов
- Уведомления

### 🔮 После MVP
- WebSocket для real-time чата
- Аналитика для заказчиков
- Интеграция с соцсетями
- Портфолио с файлами

## 📊 Разделение по уровням (USP)

1. **Новички** - Безопасное пространство для первых проектов
2. **Опытные** - Для фрилансеров с портфолио
3. **Эксперты** - Премиум конкурентные заказы

Это отличает нас от kwork.ru и upwork.com

## 💰 Монетизация

| Услуга | Цена | Назначение |
|--------|------|-----------|
| Премиум профиль | 500₽/месяц | Выделение в поиске |
| Продвижение объявления | 50-200₽ | Поднятие в топ |
| Pro подписка | 1000₽/месяц | Аналитика + экспорт |

## 🚀 Deployment

### Yandex Cloud
- **БД:** PostgreSQL в Москве (ru-central1)
- **Backend:** Serverless Functions
- **Хранилище:** Object Storage для файлов портфолио

### RuStore
- Публикация APK после MVP
