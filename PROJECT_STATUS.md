## 🔧 Технический стек

### Backend
- **Язык:** JavaScript (Node.js 18+)
- **Framework:** Express.js 4.18
- **БД:** PostgreSQL 14+ (Yandex Cloud)
- **Аутентификация:** JWT + bcryptjs
- **HTTP Client:** axios

### Frontend
- **Язык:** Dart
- **Framework:** Flutter 3.0+
- **State Management:** Provider
- **HTTP:** http package
- **Хранилище:** SharedPreferences (локальные данные)

### DevOps
- **Облако:** Yandex Cloud (Москва)
- **Платежи:** Яндекс.Касса
- **Уведомления:** Firebase Cloud Messaging
- **Платформа:** Android (RuStore)

## 🎯 MVP Функции (готово)

### ✅ Авторизация
- Регистрация с выбором типа профиля (freelancer/client)
- Вход с email и паролем
- JWT токены с 7-дневным expiration
- Сохранение токена локально

### ✅ Каталог заказов
- Загрузка списка заказов с backend
- Фильтр по уровню опыта (beginner/intermediate/expert)
- Фильтр по категории (design/programming/copywriting)
- Клик на заказ → детали (skeleton)

### ✅ Профиль пользователя
- Отображение информации профиля
- Статистика (рейтинг, кол-во завершённых заказов)
- Кнопка выхода из приложения
