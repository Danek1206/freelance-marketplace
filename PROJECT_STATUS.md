## 🚀 API ЭНДПОИНТЫ

### Auth (2)
- ✅ `POST /auth/register` — Регистрация пользователя
- ✅ `POST /auth/login` — Логин + JWT токен

### Orders (3)
- ✅ `GET /orders` — Список заказов (с фильтрами)
- ✅ `GET /orders/:id` — Деталь заказа
- ✅ `POST /orders` — Создать заказ

### Applications (2)
- ✅ `POST /applications` — Подать заявку
- ✅ `GET /orders/:id/applications` — Просмотр заявок

### Users (1)
- ✅ `GET /users/:id` — Профиль пользователя

### System (1)
- ✅ `GET /health` — Проверка здоровья сервера
